Detailed Description

template<class T>
class keyvi::dictionary::fsa::internal::MinimizationHash< T >

A memory-efficient (multi-)set that can return an "equal" object (as opposed to "same"). This consumes less memory than standard collections. When looking up a key, the key may be of a different type, but it must be possible to map it to an entry of type T. It is a multi-set in that it can contain more than one "equal" object, but it will only return the first one.

Template Parameters
Tthe type of entry.
