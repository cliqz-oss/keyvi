Detailed Description

An iterator class for matches, which makes the usage more convinient. The Lookup code simply has to implement a functor/delegate/lambda function which the iterator calls for every step

Remarks:

Most clients use C++11 lambda functions

There might be faster solutions, using 'fast delegates':
